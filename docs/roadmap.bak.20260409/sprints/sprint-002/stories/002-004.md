# 002-004: Agent recording writer and uploader

## Context
Every PTY session's output is recorded to JSONL format and uploaded to the Hub for storage and later replay. The recorder buffers chunks and uploads in batches.

## Requirements
1. `Recorder` class that attaches to a PtySession
2. Buffer output chunks and upload to Hub via WsClient at configurable interval (default 5s)
3. Send `agent:recording:upload` with chunks + `final: false`
4. On session end, flush remaining buffer and send with `final: true`
5. Handle upload failures: buffer locally, retry on next interval

## Acceptance Criteria
- [ ] Recorder captures all PTY output chunks with timestamps
- [ ] Chunks uploaded to Hub every 5s (configurable)
- [ ] Final upload sent with `final: true` on session end
- [ ] Upload failures don't crash the session (graceful retry)
