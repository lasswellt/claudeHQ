# 002-009: Agent health reporting

## Context
The agent heartbeat includes CPU and memory percentages. This story implements the system metrics collection used by the heartbeat and the `chq agent status` command.

## Requirements
1. `getSystemHealth()` function returning: cpuPercent, memPercent, diskPercent, uptime
2. CPU: calculated from `os.cpus()` over a 1-second sample
3. Memory: `1 - (os.freemem() / os.totalmem())` as percentage
4. Disk: `statvfs` or `df` for the data directory
5. Used by heartbeat interval and `chq agent status`

## Acceptance Criteria
- [ ] Returns CPU percentage (0-100) averaged across cores
- [ ] Returns memory percentage (0-100)
- [ ] Returns disk percentage for the working directory
- [ ] Values are reasonable (not always 0 or 100)
