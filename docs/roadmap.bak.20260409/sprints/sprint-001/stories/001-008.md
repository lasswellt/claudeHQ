# 001-008: WebSocket protocol ‚Äî Agent‚ÜîHub messages

## Context
All WebSocket communication between Agent and Hub uses typed, Zod-validated messages. These schemas are the single source of truth for the protocol. Per project rules, no inline message type definitions ‚Äî everything imports from @chq/shared.

## Requirements
1. Agent‚ÜíHub messages (Zod schemas + TS types):
   - `agent:register` ‚Äî machineId, version, maxSessions, os
   - `agent:heartbeat` ‚Äî machineId, activeSessions, cpuPercent, memPercent
   - `agent:session:started` ‚Äî sessionId, machineId, prompt, cwd, pid
   - `agent:session:output` ‚Äî sessionId, chunks: Array<{ts, data}>
   - `agent:session:ended` ‚Äî sessionId, exitCode, claudeSessionId
   - `agent:recording:upload` ‚Äî sessionId, chunks, final
   - `agent:queue:updated` ‚Äî machineId, queue: QueueTask[]
2. Hub‚ÜíAgent messages:
   - `hub:session:start` ‚Äî sessionId, prompt, cwd, flags
   - `hub:session:resume` ‚Äî sessionId, prompt, claudeSessionId, cwd
   - `hub:session:kill` ‚Äî sessionId
   - `hub:session:input` ‚Äî sessionId, input
   - `hub:queue:add` ‚Äî task: QueueTask
   - `hub:queue:remove` ‚Äî taskId
   - `hub:queue:reorder` ‚Äî order: string[]
3. Discriminated union schema for each direction (parse any message, get typed result)
4. Every message must have a `type` field as a string literal

## Acceptance Criteria
- [ ] `agentToHubSchema.parse(msg)` returns correctly typed discriminated union
- [ ] `hubToAgentSchema.parse(msg)` returns correctly typed discriminated union
- [ ] Unknown message types cause parse failure (exhaustive handling)
- [ ] Each message type is individually importable for handler typing
- [ ] Test: parse valid message ‚Üí correct type; parse invalid ‚Üí ZodError

## Technical Notes
- Use `z.discriminatedUnion('type', [...])` for message unions
- Each message: `z.object({ type: z.literal('agent:register'), ...fields })`
- Export individual schemas AND the union: `agentRegisterSchema`, `agentToHubSchema`
- Chunks: `z.array(z.object({ ts: z.number(), data: z.string() }))`
