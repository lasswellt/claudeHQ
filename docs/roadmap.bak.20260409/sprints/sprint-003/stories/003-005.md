# 003-005: Terminal input bar

## Context
When a session is running, the user can send PTY input (keystrokes, "yes", Ctrl+C) via a text field below the terminal.

## Requirements
1. `TerminalInput.vue` component with props: `sessionId`, `disabled`
2. VTextField with Enter to submit
3. On submit: POST `/api/sessions/:id/input` with `{ input: text + "\n" }`
4. Quick action buttons: "Yes" (sends "y\n"), "No" (sends "n\n"), "Ctrl+C" (sends "\x03")
5. Clear input after submit
6. Disabled when session is not running

## Acceptance Criteria
- [ ] Text input sends to API on Enter
- [ ] Quick action buttons send correct input
- [ ] Input clears after submit
- [ ] Disabled state when session not running

- `packages/dashboard/app/components/terminal/TerminalInput.vue` (new)
