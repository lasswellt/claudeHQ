# 003-001: Nuxt 3 + Vuetify 3 scaffold with layout

## Context
The dashboard is a Nuxt 3 SPA (ssr: false) with Vuetify 3 for UI components. This story scaffolds the app, configures Vuetify with dark/light theme, and creates the default layout (app bar, navigation drawer, main content area).

## Requirements
1. Configure Nuxt 3 with `ssr: false` for SPA mode
2. Install and configure Vuetify 3 via `vite-plugin-vuetify`
3. Install `@mdi/font` for Material Design Icons
4. Create default layout with VApp, VAppBar, VNavigationDrawer, VMain
5. Navigation items: Overview, Sessions, Machines, Queue
6. Dark/light theme toggle in app bar
7. Create placeholder pages: index.vue, machines/[id].vue, sessions/[id].vue, queues/index.vue
8. Configure `nuxt generate` to produce static output

## Acceptance Criteria
- [ ] `pnpm --filter @chq/dashboard dev` starts Nuxt dev server
- [ ] Vuetify components render (VBtn, VCard visible)
- [ ] Navigation drawer with working links
- [ ] Dark/light toggle works
- [ ] `pnpm --filter @chq/dashboard build` produces `.output/public/`
- [ ] Static files servable by Hub's @fastify/static

## Technical Notes
- Vuetify Nuxt module: use `vite-plugin-vuetify` with `vuetify-nuxt-module` or manual plugin
- Theme: `createVuetify({ theme: { defaultTheme: 'dark', themes: { dark: {...}, light: {...} } } })`
- Layout file: `packages/dashboard/app/layouts/default.vue`
- `@mdi/font` import in CSS: `@import '@mdi/font/css/materialdesignicons.css'`
- Use scoped packages: `@xterm/xterm` not `xterm` (per docs-audit research)
